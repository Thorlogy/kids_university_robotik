# Kids University Robotics (Kids Universität Robotik)

An interactive, single-file bilingual (German/English) web application designed for kids and teens (ages 8-15) to learn about robotics, artificial intelligence, sensors, kinematics, algorithms, and ethics.

## 🌟 Features
- **Bilingual Support (DE/EN)**: Switch between German and English instantly via a language toggle in the navigation bar.
- **8 Interactive Chapters**:
  1. **What is a robot?** (Sensor, Controller, Actuator & loop execution)
  2. **How does a robot move?** (Kinematics, D-Pad, Tracked/Omni drives, and interactive course)
  3. **Senses of a machine** (Ultrasonic, Camera, Lidar, and GPS interactive simulations)
  4. **Mapping & Navigation** (SLAM simulation and Roomba vacuum robot demo)
  5. **Making decisions** (If-Else thresholds and Q-Learning Gridworld training)
  6. **AI in robots** (Pixel Inspector, Convolution/Kernel workshop, Reinforcement Learning arena, and language parsing)
  7. **Real robots** (Mars Rover simulation with radio delay, Drone stabilization, and humanoid timeline from 1996 to 2026)
  8. **Humans & Ethics** (Asimov's laws, Trolley problem, Trust, Responsibility distribution, and everyday life analysis)
- **Interactive Quizzes**: A quiz at the end of each chapter with real-time feedback.
- **Certificate Generator**: Generates a personalized, canvas-rendered certificate based on quiz performance.
- **Self-Contained & Offline-Ready**: No bundlers, compilers, or external npm dependencies. Run it offline by simply opening the HTML file in any modern web browser.

## 📁 Repository Structure

* `DOKUMENTATION.md`: Technical documentation of the application architecture, design system, and interactive simulations (in German).
